Projects

WagglesBot

WagglesBot is a Discord Bot that provides users with a simple text interface to facilitate searches in Art Boorus like DerpiBooru, e621, Danbooru, and even searches within Reddit.com. Using Waggles, users can search for any tag they want and the bot sends and receives data using the chosen websites JSON API. This is my main project and contains extensive use of SQL, C#, as well as some very light bash and Linux related tooling.

DerpiGUI

DerpiGUI is a WinForms application that utilizes C#, as well as some http requests to help users mass download images from Derpibooru.org.

DeskBooru

DeskBooru is another WinForms application developed in collobaration with other students to create a media storage solution for users with large photo libraries. Using SQLite and C# it provides users an offline form of tagging images and organizing them using the design elements of an online Booru.
